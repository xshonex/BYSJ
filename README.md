

## 技术框架
采用uni-app开发，适配支持H5+小程序+APP，使用Hbuilder导入即可运行。

## 功能说明
发布资源，资源详情，资源列表，资源管理，资源购买，用户登陆，用户充值。

## 技术栈
#### 前端：vue uniapp框架
#### 后端：java springBoot
#### 管理端：vue+elementUI
#### 用户端：uniapp（vue语法）
#### 版本支持：Android+IOS+H5+小程序
#### 后台服务：SpringBoot+MyBatis+MySql
#### 核心框架：Spring Boot
#### 视图框架：Spring MVC
#### 安全框架：Apache Shiro
#### 持久层框架：MyBatis
#### 数据库连接池：Druid
#### 日志管理：Log4j
#### 定时器：Quartz
#### 数据库：MySql
